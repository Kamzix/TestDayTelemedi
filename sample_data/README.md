# Dane demonstracyjne importu pracownikow

Plik `pracownicy_demo.xlsx` sluzy do recznego testu importu pracownikow.

Zaimportuj go w aplikacji jako uzytkownik HR w widoku `Pracownicy -> Import XLSX`.

Dane sa fikcyjne i przeznaczone wylacznie do lokalnego demo. Poprawny import powinien utworzyc 3 pracownikow.

Wymagane naglowki pierwszego wiersza:

- `first_name`
- `last_name`
- `pesel`
- `birth_date`
- `identity_document`
- `email`
- `phone`
- `city`
- `street`
- `building_number`
- `apartment_number`
- `job_position`
- `active`
